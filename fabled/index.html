Fabled Buttons

Code: Jeremy Giberson
Design: Michael Schultz
Url: Fable Button Post on Forrst

Fabled buttons are based on the action buttons from the Game Fable 2. When an action was performable, an icon for the gamepad A button would show up on screen prompting the user to press and hold. The player would be required to press and hold A until the progress meter surrounding the A icon filled up. Michael Schultz was inspired by this game feature and decided to post on Forrst with a call to action on creating a similar element for HTML buttons.
